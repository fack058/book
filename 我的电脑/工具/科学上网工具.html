科学上网工具

名称 简介 Onedrive下载 来源
v2rayN 科学上网工具 🔰
nodesCatch 节点测速 🔰
CloudflareSpeedTest ip优选 🔰
batch ip优选 🔰
  • v2rayN

    适用于 Windows 的 V2Ray 客户端，支持 Xray 核心和 v2fly 核心

    如何使用

    • 如果您是新手，请从releases下载 v2rayN-Core.zip
    • 否则请下载 v2rayN.zip（您还需要将 v2ray 核心下载到与 v2rayN.exe 相同的文件夹中）
    • 运行 v2rayN.exe
  • nodesCatch

    nodesCatch V2.0 全自动一键节点测速工具

  • CloudflareSpeedTest

    🌩「自选优选 IP」测试 Cloudflare CDN 延迟和速度，获取最快 IP (IPv4 / IPv6)！另外也支持其他 CDN / 网站 IP ~

  • batch

    用于自动筛选 CF IP，并自动替换优选 IP 为 PassWall 的节点地址（WS协议）
